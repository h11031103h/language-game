# 言語化・思考力・読解力 トレーニング

言語化、思考力、読解力を鍛える3つのゲームです。

## ゲーム一覧

| 階級 | ゲーム | 説明 |
|------|--------|------|
| 初級 | 要約バトル | 文章を読んで制限字数で要約する |
| 中級 | 続きを書け | 物語の続きを文体を合わせて書く |
| 上級 | 指示の迷宮 | テキストアドベンチャー。明確な指示でゴールを目指す |

## モード

- **シングル**: 一人で練習。採点結果を確認できる
- **AB対戦**: 2人で同じ課題に挑戦し、スコアで勝敗を決める

## 起動方法

### 方法1: 開発サーバー（推奨）

```bash
npm install
npm run dev
```

**http://localhost:5174** でアクセス

### 方法2: ビルドして配信

```bash
npm run serve
```

**http://localhost:3000** でアクセス（ビルド後に自動起動）

### 方法3: プレビュー

```bash
npm run build
npm run preview
```

**http://localhost:4173** でアクセス

> ⚠️ HTMLファイルを直接開く（file://）では動作しません。必ずサーバーで配信してください。

### AB対戦も使う場合

2つのターミナルで以下を実行:

```bash
# ターミナル1: フロントエンド
npm run dev

# ターミナル2: 対戦サーバー
cd server && npm install && npm start
```

または一括起動:

```bash
npm run dev:all
```

## 技術スタック

- React 19 + TypeScript
- Vite
